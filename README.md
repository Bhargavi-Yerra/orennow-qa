# Oren Sustainability Hub — Exploratory Testing Assignment

## Overview

This repository contains my exploratory and manual testing assessment performed on **Oren Sustainability Hub (OSH)** based on the suggested reporting workflow.

The objective of this assignment was to understand the end-to-end sustainability reporting lifecycle, validate the multi-user workflow, and identify functional, usability, and workflow-related observations.

## Scope of Testing

The testing covered:

* Authentication & Access Management
* Settings & Profile Configuration
* User Management & Role Permissions
* Sustainability Report Creation (BRSR – Core)
* Question Delegation Workflow
* Collaborator Contribution Flow
* Auditor Review Process
* Dashboard & Aggregated Reporting Validation
* Multi-user collaboration and workflow sequencing
